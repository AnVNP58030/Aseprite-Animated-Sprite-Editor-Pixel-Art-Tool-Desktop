## ❗️ Troubleshooting
| Problem                        | Solution                                                                 |
|--------------------------------|--------------------------------------------------------------------------|
| Script won't run               | Run PowerShell as Administrator + `Set-ExecutionPolicy Unrestricted`     |
| Some apps remain               | Reboot and run script again (some apps reinstall on first boot)          |
| Internet issues after debloat  | Re-enable "Network Connection Broker" in the script menu                 |
| Windows Update problems        | Use "Balanced" profile or re-enable specific services                    |
| Antivirus flags the script     | Add to exclusions (common for all debloat tools)                         |
| Want to restore everything     | Run `backup.ps1` as Administrator                                        |
